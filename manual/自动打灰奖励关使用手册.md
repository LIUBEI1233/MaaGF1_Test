# 自动打灰奖励关使用手册

```!
🌟 核心功能：全自动刷取硅状带彩蛋点获取pt点和限定人形  
✅ 已集成：2星人形自动拆解系统  
```


## 🚀 使用指南

进入以下界面

1. **主菜单界面** 

   - 确认有"灰域探查"按钮
   
2. **灰域选择界面** 
   - 确认左下角显示"xx带"/"当前xxxx积分"
   - 点击运行即开始自动化流程
